# 🏥 MedTriage

A modern web-based application for managing and organizing **medical records efficiently**.  
MedTriage provides a clean, responsive interface to create, read, update, and manage patient data seamlessly.

---

## 🚀 Overview

MedTriage is designed to simplify medical record management using modern web technologies. It enables users to securely store and manage health-related data in a structured and user-friendly way.

This project demonstrates a **full-stack architecture** combining frontend UI, backend logic, and database management.

---

## ✨ Features

- 📋 **CRUD Operations**
  - Create, Read, Update, Delete medical records
- 📱 **Responsive UI**
  - Works across mobile, tablet, and desktop
- ⚡ **Fast Performance**
  - Optimized using modern frameworks
- 🎨 **Clean UI**
  - Built with Chakra UI
- 🔗 **API Ready**
  - Easily extendable with backend services

## ⚙️ Installation & Setup

### 1. Clone the Repositor

```bash
git clone https://github.com/kishalaycodes/MedTriage.git
cd MedTriage

### 2. Install Dependencies

Make sure you have **Node.js** and **Yarn** installed.

```bash
yarn install

### 3. Run Development Server

yarn dev

### 4. Open in Browser

http://localhost:3000
